배열 (array)
    여러 자료를 묶어서 활용할 수 있는 특수한 자료
    여러 개의 변수를 한번에 선언해 다룰 수 있는 자료형
    배열 내부에 들어있는 값을 요소(element)라고 한다.
    대괄호[]를 사용 , 로 구분

    각각의 요소에 접근할 때 배열 바로 뒤에 대괄호 를 입력하고 그 안에 숫자를 입력
    가장 앞에 있는 요소를 0번째 요소로 표현한다. 이때 요소의 순서를 인덱스(index)라고 한다

    -문법
    배열[인덱스]

    - 배열 요소 갯수 확인
    배열.length

    - 배열 뒷부분에 요소 추가
    배열.push(요소)

        - 인덱스를 사용해서 배열 뒷부분에 요소 추가
        const ArrayName = ['as', 'sd','sdf']

        ArrayName[10] = '한글'

        -> ArrayName = ['as', 'sd', 'sdf', empty * 7, "한글"]
        //이때 중간 7개 자리의 인덱스는 아무것도 없는 empty가 된다
        
        - length속성으로 배열 마지막 위치에 요소 추가
        ArrayName[Arrayname.length] = '추가할 요소'

        - 배열 요소 제거하기 
            인덱스 요소로 제거 -> splice()
            문법
            배열.splice(인덱스, 제거할 요소의 개수)
            ex) 배열.splice(3,2) => 배열의 인덱스[3]부터 2개의 요소를 제거([3]포함)

            값으로 요소 제거 -> indexOf()
            const 인덱스 = 배열.indexOf(요소)
            문법
            배열.splice(인덱스,1)
            //배열 내부에 해당 요고사 없을 때는 -1을 리턴한다

            + 배열 내부에서 특정 값을 가진 요소 모두 제거
            indexOf()는 하나의 요소만 지울 수 있지만 filter()메소드 를 사용하면 여러개의 요소를 지울 수 있다

            ex) const 배열이름 = ['요소1','요소2','요소3','요소4']

            에서
            배열이름.filter((item) => item !== '4')
            -> ['요소1','요소2','요소3']

        - 배열의 특정 위치에 요소 추가
